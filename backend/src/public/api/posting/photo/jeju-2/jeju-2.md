## Intro

제주도 여행에서 찍었던 사진들입니다! 다랑쉬오름이 상당히 인상 깊었습니다. 오름의 특성상 정상에 오른 뒤, 분화구를 따라서 걷게 됩니다. 이때 다랑쉬오름은 분화구 둘레의 위치에 따라 다른 느낌과, 다른 경관이 펼쳐집니다. 추후 제주도에 갈 일이 있으시다면 다랑쉬오름은 꼭 방문해보시길 추천드립니다.
